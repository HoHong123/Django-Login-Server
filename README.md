# How to setup the project
## Installation
Django install step
Python install step

## Django package management
Make ***requiremnt.txt*** file
```
$ pip freeze > name_requirement.txt
```

Install packages, using ***requirements.txt*** file
```
$ pip install -r requirements.txt
```
